## Cara Menjalankan

### 1. Pastikan Python 3.8+ terinstal
```bash
python --version
```
### 2. Install dependensi
```bash
pip install -r requirements.txt
```

### 3. Jalankan aplikasi
```bash
python app.py
```

### 4. Buka browser
```
http://localhost:5000
```
